# bootstrap-scrollTo
An animated Bootstrap scroll to anchor plugin.

## Installation

```
npm install --save bootstrap-scrollTo
```

## Development

### Dependencies
* Node.js

### Installation
Check out this repositiory. Navigate to it and run:

```
  npm install
```

### Usage

For Development:
```
npm run serve
```

For Production:
```
npm run build
```

For Deployment:
```
npm run deploy
```

To Lint:
```
npm run lint
```
or

```
npm run lint:fix
```
